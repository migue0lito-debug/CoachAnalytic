# Coach Analytic

Coach Analytic es una aplicación web enfocada en el análisis de videos deportivos, especialmente pensada para entrenadores y atletas. Permite visualizar, comparar y analizar videos de sesiones de entrenamiento para mejorar el rendimiento.

## Características principales

- **Interfaz moderna:** Diseño atractivo y adaptable a móviles y escritorio.
- **Reproductor de video:** Control de velocidad, avance/retroceso rápido, navegación por frames.
- **Modo contraste:** Comparación de dos videos en paralelo.
- **Organización**: Navegación fácil entre inicio, configuración y gestión de videos.
- **Multi-idioma:** Cambia entre español e inglés desde la interfaz.

## Instrucciones para desplegar en GitHub Pages

1. **Crea un repositorio nuevo en GitHub** (ejemplo: `coach-analytic`).
2. **Sube todos los archivos** mencionados arriba, respetando la estructura de carpetas.
3. **Activa GitHub Pages:**
    - Ve a “Settings” > “Pages”.
    - Selecciona la rama principal (`main`) y la carpeta raíz `/ (root)`.
    - Guarda los cambios.
4. **Accede a tu sitio web:**
    - La URL será algo así: `https://TU_USUARIO.github.io/coach-analytic/`  
      _(reemplaza `TU_USUARIO` por tu nombre de usuario en GitHub)_

## Uso básico

1. Abre la página principal.
2. Sube tus propios videos o usa el demo.
3. Explora los controles para analizar tus sesiones: reproduce, pausa, avanza, retrocede, ajusta la velocidad y navega por frames.
4. Usa el modo contraste para comparar dos videos al mismo tiempo.
